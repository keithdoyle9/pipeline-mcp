# Code of Conduct

This project follows the Contributor Covenant Code of Conduct version 2.1.

## Our Standards

Examples of behavior that contributes to a positive environment for this project include:

- Demonstrating empathy and respect toward other people.
- Giving and gracefully accepting constructive feedback.
- Taking responsibility, apologizing to those affected by mistakes, and learning from the experience.
- Focusing on what is best for the overall project and community.
- Showing respect for differing opinions, viewpoints, and experiences.

Examples of unacceptable behavior include:

- The use of sexualized language or imagery, and sexual attention or advances of any kind.
- Trolling, insulting or derogatory comments, and personal or political attacks.
- Public or private harassment.
- Publishing others' private information, such as a physical or email address, without explicit permission.
- Other conduct which could reasonably be considered inappropriate in a professional setting.

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may take any action they deem appropriate in response to unacceptable behavior.

## Scope

This Code of Conduct applies within all project spaces and in public spaces when an individual is officially representing the project.

## Reporting

Report violations to the maintainers through GitHub by contacting [@keithdoyle9](https://github.com/keithdoyle9). Do not use public issues for conduct reports that require privacy.

## Enforcement Guidelines

Maintainers will follow the Contributor Covenant enforcement guidelines:

1. Correction
2. Warning
3. Temporary ban
4. Permanent ban

For more details, see the full text of the Contributor Covenant at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
